Below I have details on each of the 3 projects I have done during my time in college(4th Year yet to be completed).

Dvd Selling Website - 2nd Year Project

I created a site which the user can create an account and use the site in order to view and purchase a catalogue of Dvd's.Users could create an account with the site and have their preferences be recorded so that they would be recommended Dvds based on what they bought preiously. The user could add these Dvd's to a shopping care and then go to a checkout page to purchase.






Weight Loss Application - 3rd Year Project

I created an app which the user will enter in their height, weight and age and it will calculate that persons BMI. The app allows the user to search through a food database to enter in the food they eat in other to create a food log. They can keep track of it in the app and then update their weight over time to show their progress







Gaming Matchmaker Web App - 4th Year Project

For my 4th year project I am creating an app which will match up gamers based on their skill levels in order to play games together. Users can create a forum post selecting the game and mode they wish to find people for. The app will then link them up with people looking to play the same game as them and with a similar level of skill.
